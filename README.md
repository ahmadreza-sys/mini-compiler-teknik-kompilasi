## Mata Kuliah
Teknik Kompilasi

## Deskripsi
Program mini compiler sederhana menggunakan:
- Lexer
- Parser
- AST
- TAC

Program mendukung operator:
- +
- -
- *
- /
- ^

## Contoh Input
a ^ 2 + b * c

## Contoh Output
t1 = a ^ 2
t2 = b * c
t3 = t1 + t2
